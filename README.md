# Random User Data Generator
This Python project generates random user records and writes them to a text file.  
Each record includes:
- Name
- Email
- Date of Birth
- Salary

## Features
- Generates realistic names and surnames
- Creates unique emails using random numbers
- Saves data to a `.txt` file
- Includes error handling for user inputs
- Suitable for SQL practice and testing

## Technologies
- Python
- Random module
- File handling
- Exception handling

## Usage
1. Clone the repository.
2. Run `python script_name.py`.
3. Enter the number of records required.
4. Generated records will be saved in `Datarecord.txt`.
